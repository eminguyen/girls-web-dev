HTML Introduction

HTML stands for Hypertext Markup Language.

HTML is made up of basic building blocks called elements. Elements are made up of tags and content.

A tag looks something like the following: 

        
          <p>
Basically, it is just code that comes between two arrow symbols.

Also, you should also know that most tags come in sets of two. 


        
          <p> </p>
Its buddy is called a closing tag.

Note: While most tags have a buddy, not all do. We will go over some later.

Content in most cases is just text. 


        
          <p> Example Text </p>

The above code will look produce something that looks like this:

Example Text

Notice that the tags disappear and the content stays. In this case, the content was placed between paragraph tags. These tags basically just display text on the screen plainly. Most of the content you've seen so far has been displayed using just paragraph tags (Aside from the code examples). We will explore many more types of tags soon!
